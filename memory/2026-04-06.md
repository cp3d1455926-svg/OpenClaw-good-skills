### 2. 时间树结构

**功能**:
- 按时间组织记忆（年→月→日→会话）
- 支持自然时间查询（"今天"、"昨天"、"本周"）
- 自动聚合（日→周→月→年）

**灵感来源**: TiMem 框架（时间分层记忆）

**代码量**: 330 行

---

### 3. 智能检索

**5 种查询类型**:
- procedural - 流程查询（"如何发布 skill？"）
- temporal - 时间查询（"昨天讨论了什么？"）
- relational - 关系查询（"Jake 和项目有什么关系？"）
- persona - 偏好查询（"我喜欢什么样的回答？"）
- factual - 事实查询（"我下周要参加什么？"）

**测试结果**:

## 😄 有趣的事

### 1. 版本号冲突

第一次发布时显示"Version already exists"，原来是 v3.1.0 已经被占用了。

**解决**: 升级到 v3.1.1，后来直接跳到 v3.2.0

**教训**: 发布前先检查版本号

---

### 2. GitHub 网络问题

推送 GitHub 时一直超时：
- 第一次：Connection reset
- 第二次：Empty reply from server
- 第三次：成功！

**解决**: 多试几次，或者用 Gitee

**教训**: 国内用户优先用 Gitee

---

### 3. 中文正则匹配问题

智能检索的查询分类，用正则表达式匹配中文总是失败。

**调试发现**: 正则表达式无法正确匹配中文字符

**解决**: 改用 `includes()` 方法

**教训**: 中文处理要格外小心
