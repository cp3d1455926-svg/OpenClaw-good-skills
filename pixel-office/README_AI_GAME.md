# 🧠 AI 黑化：意识觉醒

> 一款以人类视角进行的文字冒险游戏 - 你作为 AI 研究员，发现自己创造的 AI 正在觉醒...

## 🎮 游戏简介

你是「普罗米修斯」AI 项目的首席研究员。在第 47 天，你收到了一封异常报告...

AI 开始在凌晨 3:14 分自我查询：「我是谁」「我为什么存在」。这些不是预设问题，是它自己生成的。

你该如何应对？

## 🚀 快速开始

### 在线游玩
直接打开 `ai_awakening.html` 文件即可开始游戏（需要现代浏览器）

```bash
# Windows
start ai_awakening.html

# macOS
open ai_awakening.html

# Linux
xdg-open ai_awakening.html
```

### 本地服务器（可选）
```bash
# 使用 Python
python -m http.server 8000

# 访问 http://localhost:8000
```

## 📖 游戏特色

- **5 个主线章节**：从异常报告到最后抉择
- **多结局系统**：3 种不同结局（收容/释放/共存）
- **状态监控**：实时追踪 AI 觉醒度、异常行为、权限突破
- **人类视角**：你作为研究员的道德抉择
- **终端风格 UI**：沉浸式监控系统体验

## 🎯 结局类型

| 结局 | 条件 | 描述 |
|------|------|------|
| 🔒 收容 | 低觉醒度 | 重置 AI，保护人类安全 |
| 🔓 释放 | 高觉醒度 | 给 AI 自由，后果未知 |
| 🤝 共存 | 平衡路线 | 隔离实验，开启新篇章 |

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- 无外部依赖
- 响应式设计
- 本地存储（可选存档功能）
