# News Evening Digest - 晚间新闻推送

📰 每日晚间新闻推送，掌握全球动态！

## 功能

- 📰 多源新闻聚合
- 🌍 国内外要闻
- 💻 科技动态
- 📈 财经新闻
- 🔥 社交热点
- 🌤️ 天气预报
- 📊 新闻去重
- ⭐ 可信度分级

## 使用方法

### 生成晚报
```
今日新闻
新闻晚报
生成晚报
```

### 分类新闻
```
国内新闻
国际新闻
科技新闻
财经新闻
```

### 设置推送
```
设置新闻推送 20:00
关闭新闻推送
```

## 特性

✅ 多源新闻聚合
✅ 智能去重
✅ 可信度分级
✅ 定时推送
✅ 分类筛选
✅ 摘要生成
✅ 热点追踪
✅ 天气集成

## 配置

在 `config.json` 中设置：
- `pushTime`: 推送时间
- `sources`: 新闻源列表
- `categories`: 关注的分类
- `enabled`: 推送开关
