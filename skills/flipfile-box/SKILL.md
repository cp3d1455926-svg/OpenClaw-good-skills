# 翻身文件箱 - FlipFile Box

📦 智能文件管理工具，让文件"翻身"做主人！

## 功能

### 🔐 隐私保护
- 文件加密存储
- 隐私空间隔离
- 访问密码保护
- 隐藏敏感文件
- 安全删除（粉碎）
- 水印保护
- 访问日志记录
- 生物识别解锁

### 🔄 多端同步
- 云端同步
- 多设备访问
- 实时同步更新
- 冲突自动解决
- 版本历史记录
- 离线访问
- 增量同步
- 同步状态显示

### 📄 文件转换
- 50+ 格式支持
- 批量转换
- 高质量输出
- 格式自动识别
- 自定义参数
- 转换队列管理
- 进度实时显示
- 转换历史记录

## 使用方法

### 隐私保护
```
加密文件 [文件路径]
创建隐私空间
隐藏文件 [文件路径]
粉碎文件 [文件路径]
设置访问密码
```

### 多端同步
```
同步文件 [文件路径]
查看同步状态
同步历史记录
暂停同步
恢复同步
```

### 文件转换
```
转换文件 file.docx 转 pdf
批量转换 文件夹路径 转 pdf
图片转格式 photo.jpg 转 png
视频转 MP4 video.avi
PDF 转 Word file.pdf
```

## 配置

在 `config.json` 中设置：
- `encryptAlgo`: 加密算法（AES-256）
- `syncInterval`: 同步间隔（分钟）
- `cloudProvider`: 云服务商
- `convertQuality`: 转换质量
- `autoBackup`: 自动备份开关
