# QR Code Generator - 二维码生成器

📱 快速生成和识别二维码，支持多种类型！

## 功能

- 🔗 网址转二维码
- 📝 文本转二维码
- 📇 名片二维码
- 📶 WiFi 二维码
- 📱 小程序码生成
- 🖼️ 自定义样式
- 📸 二维码识别
- 🎨 添加 Logo

## 使用方法

### 生成二维码
```
生成二维码 https://example.com
二维码 https://github.com
生成 WiFi 二维码 名称：MyWiFi 密码：123456
```

### 识别二维码
```
识别二维码 [上传图片]
解析二维码 [图片链接]
```

### 自定义样式
```
生成二维码 https://example.com 带 Logo
二维码 文本：你好世界 颜色：蓝色
```

## 特性

## 配置

在 `config.json` 中设置：
- `defaultSize`: 默认尺寸（300）
- `defaultFormat`: 输出格式（PNG）
- `logoPath`: 默认 Logo 路径
- `errorCorrection`: 容错级别（M）
