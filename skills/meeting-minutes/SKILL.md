# Meeting Minutes - 会议纪要生成器

📝 自动生成会议纪要，提升会议效率！

## 功能

- 🎤 语音转文字
- 📝 自动总结要点
- ✅ 提取待办事项
- 👥 识别发言人
- 📅 提取时间信息
- 📊 生成会议报告
- 🔍 关键词提取
- 📧 邮件发送纪要

## 使用方法

### 创建纪要
```
创建会议纪要 [上传录音]
会议纪要 [会议记录文本]
```

### 提取待办
```
提取待办事项 [会议记录]
会议任务分配 [会议记录]
```

### 生成报告
```
生成会议报告 [会议记录]
会议总结 [会议记录]
```

### 发送邮件
```
发送会议纪要 参会者邮箱
```

## 特性

✅ 支持多种音频格式
✅ 自动识别发言人
✅ 智能提取关键信息
✅ 待办事项追踪

## 配置

在 `config.json` 中设置：
- `template`: 纪要模板
- `autoEmail`: 自动发送邮件
- `taskSystem`: 任务系统对接
- `language`: 识别语言
