# 📖 Story Game - 故事接龙

AI 互动讲故事、创意写作的娱乐工具。

## 功能列表

- ✅ 故事开头生成
- ✅ 轮流接龙
- ✅ 剧情转折
- ✅ 角色生成
- ✅ 场景生成
- ✅ 随机故事元素

## 使用示例

```
开始故事：start 奇幻冒险
继续：continue 小明走进了森林
添加转折：twist
新角色：character
新场景：location
随机元素：random
```

## 玩法说明

1. 一人开头（或用 random 生成）
2. 轮流接龙
3. 可以添加转折/角色/地点
4. 共同创作完整故事
