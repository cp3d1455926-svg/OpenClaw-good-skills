# Data Visualizer - 数据可视化工具

📊 一键生成精美图表，让数据会说话！

## 功能

- 📈 柱状图/条形图
- 📉 折线图/面积图
- 🥧 饼图/环形图
- 📊 散点图/气泡图
- 🗺️ 地图/热力图
- 📋 数据透视表
- 🎨 自定义样式
- 📱 交互式图表

## 使用方法

### 创建图表
```
生成柱状图 [数据]
折线图 [数据]
饼图 [数据]
```

### 数据导入
```
导入 Excel 数据 file.xlsx
导入 CSV data.csv
导入 JSON data.json
```

### 自定义样式
```
生成图表 [数据] 样式：蓝色系
柱状图 [数据] 标题：销售统计
```

### 导出图表
```
导出图表 PNG
导出图表 SVG
导出图表 PDF
```

## 配置

在 `config.json` 中设置：
- `defaultType`: 默认图表类型
- `theme`: 默认主题
- `outputDir`: 输出目录
- `interactive`: 交互模式
