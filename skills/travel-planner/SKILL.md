# Travel Planner - 旅行规划助手

✈️ 帮你规划完美旅程，从行程到预算一站式服务！

## 功能

- 🗺️ 智能行程规划
- 🏨 酒店推荐
- 🍽️ 美食推荐
- 💰 预算估算
- 📋 行李清单
- 🌤️ 天气提醒

## 使用方法

### 创建行程
```
规划旅行 北京 3 天 5000 元
规划旅行 日本 7 天 2 万 亲子游
```

### 查询信息
```
北京攻略
日本签证
旅行清单
```

### 修改行程
```
调整行程 增加故宫
调整行程 减少购物
```

## 特性

✅ 根据预算智能推荐
✅ 考虑季节和天气
✅ 交通/住宿/餐饮/景点全覆盖
✅ 亲子/情侣/独自旅行等场景
✅ 实时汇率转换
✅ 紧急联系方式
