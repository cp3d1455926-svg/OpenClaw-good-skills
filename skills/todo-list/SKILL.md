# 📝 Todo List - 待办事项管理

智能待办事项管理技能，支持 GTD 方法、优先级排序、智能提醒。

## 功能列表

- ✅ 添加/删除/完成任务
- ✅ 设置优先级（高/中/低）
- ✅ 分类管理（工作/学习/生活/其他）
- ✅ 截止日期提醒
- ✅ 每日计划生成
- ✅ 完成情况统计

## 使用示例

```
添加任务：明天下午 3 点开会，优先级高，分类工作
查看任务：显示今天的待办
完成任务：完成任务 1
删除任务：删除任务 2
统计：本周完成了多少任务
```

## 数据格式

```json
{
  "tasks": [
    {
      "id": 1,
      "title": "明天下午 3 点开会",
      "priority": "high",
      "category": "work",
      "due_date": "2026-03-21T15:00:00",
      "completed": false,
      "created_at": "2026-03-20T21:00:00"
    }
  ]
}
```

## 优先级说明

- 🔴 high - 紧急重要，优先处理
- 🟡 medium - 重要不紧急
- 🟢 low - 不紧急不重要

## 分类说明

- 💼 work - 工作相关
- 📚 study - 学习相关
- 🏠 life - 生活相关
- 📦 other - 其他
