# ⏱️ Time Tracker - 时间记录

番茄工作法、时间块记录、效率分析的时间管理工具。

## 功能列表

- ✅ 开始/结束时间记录
- ✅ 番茄工作法支持
- ✅ 今日时间统计
- ✅ 按类型分类（工作/休息）
- ✅ 效率分析报告

## 使用示例

```
开始记录：start 写代码
番茄工作：pomodoro
结束记录：end
查看统计：stats
```

## 数据类型

- work - 工作时间
- break - 短休息（5 分钟）
- long_break - 长休息（15-30 分钟）
