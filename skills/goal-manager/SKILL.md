# Goal Manager - 目标管理助手

🎯 帮你设定、追踪和达成人生目标！

## 功能

- 🎯 SMART 目标设定
- 📋 目标分解为任务
- 📊 进度追踪
- ⏰ 截止日期提醒
- 🏆 成就系统
- 📈 数据分析报表

## 使用方法

### 设定目标
```
设定目标 减肥 10 斤 6 个月
设定目标 学 Python 3 个月
设定目标 存钱 5 万 1 年
```

### 追踪进度
```
更新进度 减肥 3 斤
我的目标
本周完成
```

### 目标管理
```
目标详情 [目标名]
删除目标 [目标名]
目标复盘
```

## 特性

✅ SMART 原则目标设定（具体/可衡量/可实现/相关性/时限性）
✅ 大目标自动分解为小任务
✅ 进度可视化（百分比/图表）
✅ 定期复盘提醒
✅ 成就徽章系统
✅ 目标完成度分析
