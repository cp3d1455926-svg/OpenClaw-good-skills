# Expense Tracker - 记账助手

💰 智能记账助手，帮你管理日常开支！

## 功能

- 📝 快速记录收支
- 📊 分类统计报表
- 🎯 预算管理
- 📈 消费趋势分析
- 💡 智能消费建议
- 🔄 多币种支持

## 使用方法

### 记录收支
```
记账 -50 午餐
记账 +3000 工资
记账 -200 购物 分类：娱乐
```

### 查询统计
```
今日支出
本月支出
分类统计
预算情况
```

### 预算管理
```
设置预算 3000
预算提醒 80%
```

## 特性

✅ 支持中文分类（餐饮/交通/购物/娱乐/学习/医疗/住房/其他）
✅ 自动识别收支类型（+收入/-支出）
✅ 月度/年度报表生成
✅ 消费趋势图表（文本版）
✅ 超预算提醒
✅ 数据导出（CSV/JSON）

## 示例输出

```
💰 今日记账 #2024-03-21

✅ 支出：-50.00 元 - 午餐（餐饮）
✅ 支出：-20.00 元 - 地铁（交通）
✅ 收入：+3000.00 元 - 工资

━━━ 今日统计 ━━━
📊 总支出：70.00 元
💵 总收入：3000.00 元
💰 净收入：2930.00 元

📈 本月剩余预算：2430/3000 元 (81%)
```
