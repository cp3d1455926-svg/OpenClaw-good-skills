# Weather Pro - 专业天气预报

🌤️ 专业级天气预报助手，提供精准天气信息！

## 功能

- 🌡️ 实时天气查询
- 📅 7 天天气预报
- ⚠️ 天气预警提醒
- 🌍 多城市支持
- 📊 详细气象数据
- 🌅 日出日落时间
- 💨 空气质量指数
- 🌧️ 降水概率预测

## 使用方法

### 查询天气
```
天气 北京
天气 上海
weather Beijing
```

### 多城市对比
```
天气对比 北京 上海 广州
```

### 预警信息
```
天气预警 深圳
```

### 详细预报
```
详细天气 杭州
7 天预报 成都
```

## 特性

✅ 支持全球主要城市
✅ 实时更新天气数据

## 配置

在 `config.json` 中设置：
- `defaultCity`: 默认城市
- `unit`: 温度单位（celsius/fahrenheit）
- `alertEnabled`: 预警开关
- `updateInterval`: 更新间隔（分钟）
