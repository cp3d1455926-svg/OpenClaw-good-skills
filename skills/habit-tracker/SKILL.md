# Habit Tracker - 习惯养成助手

🔥 帮你建立好习惯，戒除坏习惯！

## 功能

- ✅ 习惯打卡
- 📊 连续天数统计
- 📈 习惯养成曲线
- 🏆 成就徽章
- 💡 习惯建议
- 👥 习惯挑战

## 使用方法

### 创建习惯
```
创建习惯 早起 每天 7:00
创建习惯 运动 每周一三五
创建习惯 戒烟 一次性
```

### 打卡
```
打卡 早起
打卡 运动
今日打卡
```

### 统计
```
习惯统计
连续天数 早起
习惯报告
```

## 特性

✅ 支持多种频率（每天/每周 X 天/每月 X 次）
✅ 连续打卡统计（Streak）
✅ 习惯养成曲线（21/66 天理论）
✅ 成就徽章系统
✅ 习惯挑战（7 天/21 天/66 天）
✅ 失败原谅机制（每月 3 次机会）
