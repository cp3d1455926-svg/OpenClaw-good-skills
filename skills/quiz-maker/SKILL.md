# 📝 Quiz Maker - 测试生成器

知识测验、答题闯关、自动评分的学习工具。

## 功能列表

- ✅ 按分类生成测验
- ✅ 自定义题目数量
- ✅ 多种难度选择
- ✅ 自动评分
- ✅ 测试记录统计
- ✅ 每日练习

## 使用示例

```
开始测验：start Python 5
答题：answer 1 A
查看分类：categories
每日练习：practice
查看统计：stats
```

## 题目分类

- Python
- 常识
- 英语
- 数学
- 科技
- 历史

## 难度等级

- easy - 简单
- medium - 中等
- hard - 困难
