## 🚀 快速开始

### 方式一：网页版（推荐）

直接双击打开：
```bash
# 在文件管理器中双击 index.html
openclaw-pixel-office/index.html
```

或者用浏览器打开：
```bash
# Chrome/Edge/Firefox 打开
start index.html
```

### 方式二：Python 终端版

```bash
cd openclaw-pixel-office/python
python office.py
```

选择运行模式：
1. **终端版** - 彩色终端界面
2. **网页版** - 启动本地服务器
3. **浏览器** - 直接打开网页版

### 方式三：本地服务器

```bash
cd openclaw-pixel-office
python -m http.server 8080

# 访问 http://localhost:8080
```

## 🎨 自定义

### 添加新技能

编辑 `data.js`，在 `SKILLS` 对象中添加：

```javascript
"your-skill": {
  name: "你的技能名",
  icon: "🎯",
  version: "v1.0",
  category: "分类",
  codeSize: "10KB",
  description: "技能描述",
  features: ["功能 1", "功能 2"],
  commands: ["使用示例 1", "使用示例 2"]
}
```

### 修改颜色主题

编辑 `style.css` 中的 CSS 变量：

```css
:root {
  --accent-color: #00ff88;  /* 主色调 */
  --v2-color: #ffd700;      /* v2.0 标识色 */
  --bg-color: #2d2d44;      /* 背景色 */
}
```

### 更换主角

修改 `index.html` 中的龙虾 ASCII 艺术：

```html
<div class="pixel-lobster" id="lobster">
  <!-- 在这里替换成你的像素艺术 -->
</div>
```
