# 📈 Stock Watcher - 股票观察助手

## 触发规则

**关键词：**
- 股票、股价、行情
- 涨跌幅、涨停、跌停
- 上证指数、纳斯达克
- 基金、美股、港股

**场景：**
- 用户查询股票价格
- 用户设置股价提醒
- 用户想看市场行情

## 功能描述

### 核心功能

1. **股价查询**
   - A 股/港股/美股
   - 实时股价
   - 涨跌幅

2. **自选股管理**
   - 添加自选
   - 移除自选
   - 分组管理

3. **价格提醒**
   - 涨跌幅提醒
   - 目标价提醒
   - 新闻提醒

4. **市场分析**
   - 大盘行情
   - 板块涨跌
   - 财经新闻

## 权限需求

- [x] 网络访问（股票 API）
- [x] 文件读写（自选股存储）
- [ ] 定时任务（价格提醒）
