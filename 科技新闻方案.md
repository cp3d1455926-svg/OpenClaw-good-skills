# 科技晚报 - 真实新闻搜索方案

## 方案一：手动触发（推荐）
每天晚上19点，你给我发消息：
```
搜索今日科技新闻
```

我会用 Tavily 搜索真实新闻，然后整理成小红书文案发给你。

## 方案二：定时任务 + 手动确认
设置定时任务提醒我，每天晚上19点我主动搜索，然后把结果发给你确认。

## 方案三：半自动生成
我创建一个脚本，每天晚上：
1. 随机选择一个科技主题
2. 生成新闻文案框架
3. 你提供真实新闻素材
4. 我整合成完整文案

## 当前可用工具

### Tavily 搜索（已配置）
- 可以搜索真实新闻
- 需要手动触发
- 支持中文搜索

### 使用方法
给我发消息：
```
搜索：人工智能最新动态
```
或
```
帮我找今天AI领域的新闻
```

我会搜索并整理成小红书风格。
