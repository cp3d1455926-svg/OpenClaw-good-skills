## 📖 功能说明

价格监控、历史价格查询、降价提醒、多平台比价。支持京东、淘宝、拼多多等主流电商平台。

### 核心功能
- 🔍 **价格查询** - 实时查询商品价格
- 📊 **历史价格** - 90 天价格走势图
- 💡 **购买建议** - 基于价格趋势的智能建议
- 🔍 **多平台比价** - 京东/天猫/拼多多比价
- 📉 **降价提醒** - 降价自动推送
- 📋 **监控列表** - 管理监控的商品

## 🔧 配置说明

### 缓存配置

- 缓存文件：`price_cache.json`
- 缓存过期：30 分钟
- 自动重试：最多 3 次

### 备用数据

当电商平台 API 不可用时，自动切换到备用数据：

- 6 款热门商品（iPhone/MacBook/耳机等）
- 历史价格数据（90 天）
- 多平台价格对比

---

## 💡 购买建议说明

系统根据历史价格自动给出购买建议：

| 情况 | 建议 |
|------|------|
| 当前价 > 平均价 10% | ❌ 价格偏高，建议等等 |
| 当前价 < 平均价 10% | ✅ 价格较低，值得入手 |
| 当前价 ≈ 历史最低 | ✅ 接近最低价，可以入手 |
| 其他 | 😐 价格中等，不急可等 |

## ⚠️ 注意事项

1. **拼多多价格** - 由于反爬较严，使用备用数据
2. **价格实时性** - 缓存 30 分钟，可能导致价格延迟
3. **电商平台** - 可能反爬，建议用官方 API
4. **数据准确性** - 价格数据仅供参考，以实际为准
