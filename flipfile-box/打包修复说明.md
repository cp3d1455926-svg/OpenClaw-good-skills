# 翻身文件箱 - 打包修复

## ⚠️ 问题说明

打包时遇到编码问题，需要手动修复。

## 🔧 解决方案

### 方法 1: 使用简化版本

我已经创建了一个稳定版本 `main.py`，用它打包：

```powershell
cd C:\Users\shenz\.openclaw\workspace\flipfile-box

# 删除旧版本
Remove-Item build,dist,*.spec -Recurse -Force -ErrorAction SilentlyContinue

# 用 main.py 打包
C:\Users\shenz\AppData\Roaming\Python\Python314\Scripts\pyinstaller.exe --onefile --windowed --name "翻身文件箱" main.py
```

### 方法 2: 手动修复 main_v2.py

1. 用文本编辑器打开 `main_v2.py`
2. 查找所有 `self.style.COLORS`
3. 替换为 `self.color_scheme.COLORS`
4. 保存为 UTF-8 编码
5. 重新打包
