## 🔄 真实文件转换

### 已支持的转换

#### 📷 图片转换（需要 Pillow）
```bash
pip install Pillow
```

**支持格式**:
- JPG ↔ PNG ↔ GIF ↔ WebP ↔ BMP ↔ TIFF

**示例**:
```python
python converter.py photo.jpg png 95
```

#### 📄 文档转换（需要 LibreOffice）
**安装 LibreOffice**: https://www.libreoffice.org/download/

**支持格式**:
- Word (.doc/.docx) ↔ PDF
- Excel (.xls/.xlsx) ↔ PDF
- PowerPoint (.ppt/.pptx) ↔ PDF
- TXT, RTF, ODT

#### 🎬 视频转换（需要 FFmpeg）
**安装 FFmpeg**: https://ffmpeg.org/download.html

**支持格式**:
- MP4 ↔ AVI ↔ MOV ↔ MKV ↔ WMV ↔ FLV

## 🎯 功能使用

### 🔐 文件加密
1. 点击左侧 "🔐 隐私保护"
2. 点击 "📁 加密文件"
3. 选择文件，输入密码（至少 8 位）
4. 生成 `.encrypted` 文件

### 🔄 文件转换
1. 点击左侧 "📄 文件转换"
2. 点击 "浏览" 选择文件
3. 选择目标格式（PNG/JPG/PDF 等）
4. 点击 "🔄 开始转换"

### 📊 查看转换结果
```
✅ 文件已转换
📄 输出路径：C:\...\photo.png
📊 文件大小：1.2MB
```

## ⚠️ 注意事项

### 转换失败原因
1. **缺少依赖库** - 安装对应的库
2. **格式不支持** - 检查格式列表
3. **文件损坏** - 检查原文件
4. **权限不足** - 以管理员身份运行

### 推荐配置
- **Python**: 3.8+
- **内存**: 4GB+
- **硬盘**: 1GB+ 可用空间
