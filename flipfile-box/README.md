# 翻身文件箱 - FlipFile Box

📦 智能文件管理桌面软件

## 功能

### 🔐 隐私保护
- 文件加密/解密
- 隐私空间
- 密码保护
- 文件粉碎
- 隐藏文件

### 🔄 多端同步
- 云端同步
- 多设备访问
- 版本历史
- 冲突解决

### 📄 文件转换
- 50+ 格式
- 批量转换
- 高质量输出

## 技术栈

- **GUI**: PyQt5 / Tkinter
- **加密**: cryptography (AES-256)
- **同步**: 云存储 API
- **转换**: 各格式转换库
