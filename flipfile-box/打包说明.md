# 翻身文件箱 - 打包说明

## 📦 打包成 EXE

### 方法 1: 一键打包（推荐）

**步骤**:
1. 双击 `快速打包.bat`
2. 等待打包完成（约 1-2 分钟）
3. 在当前目录找到 `翻身文件箱.exe`

### 方法 2: 完整打包（带图标）

**步骤**:
1. 准备图标文件 `icon.ico`（可选）
2. 双击 `打包.bat`
3. 等待打包完成

---

## 📋 打包前准备

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 安装打包工具
```bash
pip install pyinstaller
```

---

## 🎯 打包选项

### 基础打包
```bash
pyinstaller --onefile --windowed main_v2.py
```

### 带图标打包
```bash
pyinstaller --onefile --windowed --icon=icon.ico main_v2.py
```

### 完整打包（包含所有文件）
```bash
pyinstaller --onefile --windowed ^
    --name "翻身文件箱" ^
    --icon=icon.ico ^
    --add-data "converter.py;." ^
    --hidden-import PIL ^
    main_v2.py
```

## ⚠️ 常见问题

### Q1: 打包失败
**原因**: 缺少依赖库
**解决**: 
```bash
pip install Pillow pyinstaller
```

### Q2: 杀毒软件报毒
**原因**: PyInstaller 打包的误报
**解决**: 
- 添加到杀毒软件白名单
- 使用数字签名（需要证书）

### Q3: 运行闪退
**原因**: 缺少运行时库
**解决**: 
- 安装 Visual C++ Redistributable
- 查看日志：运行 `翻身文件箱.exe --console`

### Q4: 文件太大
**解决**: 
- 使用 `--onefile` 单文件模式
- 移除不必要的库
- 使用 UPX 压缩

## 🎨 添加图标

### 方法 1: 使用在线工具
1. 访问：https://www.icoconverter.com/
2. 上传图片（PNG/JPG）
3. 下载 ICO 格式
4. 重命名为 `icon.ico`
5. 运行 `打包.bat`

### 方法 2: 使用 Photoshop
1. 新建 256x256 画布
2. 设计图标
3. 保存为 PNG
4. 用在线工具转 ICO

## 💡 优化建议

### 减小程序体积
1. 使用 `--onefile` 单文件
2. 移除未使用的库
3. 使用 UPX 压缩：
```bash
pyinstaller --onefile --upx-dir=upx main_v2.py
```

### 加快启动速度
1. 使用 `--onedir` 目录模式（启动更快）
2. 避免导入大型库

### 提高兼容性
1. 在目标系统相同版本的 Windows 上打包
2. 使用 32 位 Python 打包（兼容 32/64 位系统）
