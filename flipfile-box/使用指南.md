# 翻身文件箱 - 使用指南

## 📦 软件介绍

**翻身文件箱** 是一款智能文件管理桌面软件，集隐私保护、多端同步、文件转换于一体。

---

## 🚀 快速开始

### 1. 安装依赖

```bash
cd flipfile-box
pip install -r requirements.txt
```

### 2. 启动软件

**方法 1: 双击启动**
```
双击 "启动.bat"
```

**方法 2: 命令行启动**
```bash
python main.py
```

---

## 🎯 功能使用

### 🔐 隐私保护

#### 加密文件
1. 点击 "🔐 隐私保护"
2. 点击 "📁 加密文件"
3. 选择要加密的文件
4. 输入密码（至少 8 位）
5. 生成 `.encrypted` 加密文件

#### 解密文件
1. 点击 "🔓 解密文件"
2. 选择 `.encrypted` 文件
3. 输入密码
4. 生成解密后的原文件

#### 创建隐私空间
1. 点击 "🗄️ 创建隐私空间"
2. 选择存储位置
3. 设置访问密码
4. 生成隐藏文件夹

#### 粉碎文件
1. 点击 "🗑️ 粉碎文件"
2. 选择要删除的文件
3. 确认删除
4. 文件被彻底粉碎（不可恢复）

---

### 🔄 多端同步

#### 查看同步状态
1. 点击 "🔄 多端同步"
2. 查看各设备同步状态
3. 点击 "🔄 立即同步"

#### 配置云存储
编辑 `config.json`:
```json
{
  "cloudProvider": "auto",
  "syncInterval": 5,
  "storagePath": "D:\\CloudStorage"
}
```

---

### 📄 文件转换

#### 单个文件转换
1. 点击 "📄 文件转换"
2. 点击 "浏览" 选择文件
3. 选择目标格式（pdf/docx/jpg/png/mp4/txt）
4. 点击 "🔄 开始转换"
5. 等待转换完成

#### 支持的格式
- **文档**: Word, PDF, TXT
- **图片**: JPG, PNG, GIF, WebP
- **视频**: MP4, AVI, MOV

## ⚙️ 设置说明

### 常规设置
- **默认加密算法**: AES-256
- **同步间隔**: 5 分钟
- **云服务商**: 自动选择
- **转换质量**: 90%
- **自动备份**: 开启

### 修改设置
1. 点击 "⚙️ 设置"
2. 修改相应参数
3. 点击 "💾 保存设置"

## 🔒 安全提示

### 密码安全
✅ 使用强密码（大小写 + 数字 + 符号）
✅ 定期更换密码
✅ 不要分享密码
✅ 启用双重验证

### 文件安全
✅ 定期备份加密文件
✅ 不要在公共电脑使用
✅ 使用文件粉碎功能删除敏感文件
✅ 隐藏隐私空间文件夹

## 📊 常见问题

### Q: 忘记密码怎么办？
A: 加密文件无法恢复，请妥善保管密码。

### Q: 同步失败怎么办？
A: 检查网络连接和云存储配置。

### Q: 转换失败怎么办？
A: 检查文件格式是否支持，或尝试其他转换工具。
