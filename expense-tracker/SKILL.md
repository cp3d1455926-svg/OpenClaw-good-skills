# 💰 Expense Tracker - 记账助手

## 触发规则

**关键词：**
- 记账、花钱、支出
- 收入、工资、奖金
- 预算、消费分析、花了多少钱

**场景：**
- 用户想记录日常开销
- 用户查询消费统计
- 用户需要预算管理

## 功能描述

### 核心功能

1. **快速记账**
   - 记录支出/收入
   - 自动分类
   - 添加备注

2. **消费统计**
   - 每日/周/月统计
   - 分类统计
   - 趋势分析

3. **预算管理**
   - 设置月度预算
   - 超支提醒
   - 预算执行率

4. **账单查询**
   - 按日期查询
   - 按分类查询
   - 关键词搜索

## 权限需求

- [x] 文件读写（账单存储）
- [ ] 定时任务（预算提醒）
