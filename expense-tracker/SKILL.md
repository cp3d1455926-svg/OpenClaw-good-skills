# 💰 Expense Tracker v2.0 - 记账助手

## 触发规则

**关键词：**
- 记账、花钱、支出、收入
- 统计、花了多少钱、报表
- 预算、超支
- 储蓄、存款、攒钱
- 消费分析、分类

**场景：**
- 用户想记录日常收支
- 用户查询消费统计
- 用户设置和查看预算
- 用户管理储蓄目标
- 用户分析消费习惯

## 功能描述

### 核心功能

1. **记账系统**
   - 快速记账（自动识别分类）
   - 收入/支出分类
   - 20+ 子分类支持
   - 备注和时间记录

2. **分类体系**（10 大类，20+ 子分类）
   - 🍜 餐饮（早餐/午餐/晚餐/外卖/咖啡奶茶/零食/聚餐）
   - 🚗 交通（地铁公交/打车/加油/停车/保养/保险）
   - 🛍️ 购物（衣服/鞋子/数码/日用品/化妆品/家居）
   - 🎬 娱乐（电影/游戏/聚会/KTV/运动/演出）
   - 🏠 居住（房租/水电/物业/网费/维修/清洁）
   - 🏥 医疗（看病/买药/体检/保健品/健身）
   - 📚 学习（书籍/课程/培训/考试/订阅）
   - 🧧 人情（红包/礼物/请客/捐款）
   - 📱 通讯（话费/流量/宽带）
   - 📦 其他（杂项）

3. **统计报表**
   - 周/月/年统计
   - 分类统计（金额/占比/笔数）
   - 预算执行率
   - 每日趋势
   - 收支对比

4. **预算管理**
   - 月度总预算
   - 分类预算限额
   - 超支预警（80% 提醒，100% 警告）
   - 预算执行报告

5. **储蓄目标**（6 个模板）
   - 应急基金（6 个月生活费）
   - 旅行基金
   - 购房首付
   - 购车计划
   - 教育基金
   - 创业基金

6. **消费分析**
   - 最大支出分类识别
   - 日均消费计算
   - 省钱建议
   - 预算超支提醒

## 命令语法

- `花了 [金额] 元 [分类]` - 记录支出
- `收入 [金额] 元` - 记录收入
- `这个月花了多少钱` - 月度统计
- `本周统计` / `年度报表` - 周期统计
- `设置每月预算 [金额]` - 设置预算
- `查看预算` - 预算报告
- `创建 [储蓄目标名]` - 创建储蓄目标
- `更新 [储蓄目标名] [金额]` - 更新储蓄进度
- `消费分析` - 消费分析
- `有哪些分类` - 分类列表
