# ✈️ Travel Planner v2.0 - 旅行规划助手

## 触发规则

**关键词：**
- 旅行规划、旅游计划、行程安排
- 去哪里玩、推荐景点、旅游攻略
- 预算估算、旅行费用
- 打包清单、行李准备
- 旅行贴士、注意事项

**场景：**
- 用户计划旅行需要行程建议
- 用户查询目的地景点
- 用户需要预算估算
- 用户询问打包清单
- 用户想了解旅行注意事项

## 功能描述

### 核心功能

1. **行程规划**
   - 根据天数智能规划行程
   - 景点路线优化
   - 时间分配建议
   - 用餐推荐

2. **景点推荐**
   - 12 个热门城市数据库
   - 热门景点 + 小众景点
   - 景点类型标签（历史/自然/文化/乐园等）
   - 季节性推荐

3. **预算估算**
   - 详细预算明细（餐饮/住宿/门票/交通/购物）
   - 人均预算计算
   - 不同旅行模板预算系数

4. **旅行模板**（10+ 类型）
   - 情侣游、亲子游、背包客、豪华游
   - 摄影游、美食游、文化游、冒险游
   - 养生游、毕业旅行

5. **打包清单**
   - 通用清单 + 季节清单
   - 特殊需求清单（摄影/商务/海边）
   - 个性化生成

6. **旅行贴士**
   - 最佳旅行季节
   - 注意事项
   - 当地特色建议

## 使用示例

### 基础行程规划
```
用户：帮我规划 3 天的北京旅行
助手：✈️ 北京 3 日游

📅 Day 1
- 故宫 (4 小时，¥60)
- 天安门广场 (2 小时，免费)
🍽️ 用餐：早餐酒店附近，午餐东城区，晚餐王府井

📅 Day 2
- 颐和园 (4 小时，¥30)
- 圆明园 (3 小时，¥25)

📅 Day 3
- 八达岭长城 (6 小时，¥40)

💰 预算：总计¥1750（人均）
```

### 旅行模板
```
用户：情侣游去三亚 5 天
助手：✈️ 三亚 5 日游 (浪漫双人之旅)

📅 行程安排...
💰 预算：总计¥6500（人均¥3250）
💡 贴士：预订景观房，安排烛光晚餐
```

```

□ 手机 + 充电器
□ 防晒霜
□ 泳衣
□ 太阳镜

### 旅行贴士
```

## 命令语法

- `规划 [天数] 天的 [目的地] 旅行` - 创建行程
- `[模板] 去 [目的地] [天数] 天` - 使用模板
- `[目的地] 要多少钱` - 预算查询
- `去 [目的地] 带什么` - 打包清单
- `[目的地] 注意事项` - 旅行贴士
- `推荐去哪里玩` - 目的地推荐
- `有哪些旅行模板` - 模板列表
